A full-stack **Uber Clone** that simulates a real-world ride-hailing platform with rider and driver registration, real-time ride booking, live location tracking, and fare calculation.

---

## Features

- Rider and Driver Authentication (Signup/Login)
- Real-time Ride Booking
- Live Location Tracking using Map APIs
- Automated Fare Calculation
- Ride History for Riders and Drivers
- Responsive User Interface

---

## Tech Stack

- **Frontend:** React.js / Angular
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **APIs:** Google Maps / Geo-Location APIs
- **Real-time Communication:** Socket.io
